# Budget Tracker App

A personal budget management application built with React.js.

## Features
- **Manage Budget**: Set and track your total budget.
- **Track Expenses**: Add new expenses with names and costs.
- **Real-time Calculations**: Automatically updates remaining balance and total spent.
- **Expense History**: View a list of all your expenses.
- **Delete Expenses**: Easily remove entries from your list.

## Tech Stack
- **Frontend**: React.js
- **Styling**: Bootstrap & CSS
- **Icons**: React Icons
- **UUID**: Unique ID generation for expenses

## Getting Started

### Prerequisites
- Node.js installed on your machine.

### Installation
1. Clone the repository (or download the source).
2. Run `npm install` to install dependencies.

### Running the App
1. Run `npm start` to start the development server.
2. Open [http://localhost:3000](http://localhost:3000) in your browser.
